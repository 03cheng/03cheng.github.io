Android信息存储之-JSON

发表于 2015-03-28   |  

前言

Android 存储数据有很多种方法，它们各自有各自的好处，JSON 就是其中一种。
JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式，易于阅读和编写，同时也易于机器解析和生成。
我在经过 SQLite 数据库存储信息的学习之后，再学习 JSON 存储信息发现更加简单了。
与大家分享一下我的学习过程。

导入 Gson 包
